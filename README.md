# 🎤 Voice Desktop Assistant

A Python-based Voice Desktop Assistant that performs tasks using voice commands. It helps users interact with their system hands-free.

---

**Features**

-  Speech Recognition
-  Text-to-Speech Output
-  Open Websites (Google, YouTube)
-  Open Applications
-  Tell Time & Date
-  Search on Internet



**Tech Stack**

- Python
- SpeechRecognition
- pyttsx3
- OS
- Webbrowser
- Datetime
  
How It Works

The assistant listens to your voice, converts it into text, processes the command, and performs actions like opening apps or searching the web.
